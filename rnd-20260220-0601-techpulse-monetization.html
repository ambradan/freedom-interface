  • Enterprise (custom): API access, white-label, dedicated LLM

2. Data-as-a-Service (DaaS)

Vendere insights aggregati e anonimi:

  • Market Reports: $499-2,999 per report verticale (es. "AI Engineers Demand Q1 2026")
  • API Credits: $0.10 per skill prediction, $0.50 per talent forecast
  • Benchmark Access: subscription annuale $5k-50k per comparazione settoriale

3. Embedded Intelligence (B2B2C)

White-label per ATS, HR platforms, recruitment agencies:

  • Licensing fee: $2k-10k/mo + revenue share 15-25%
  • Esempi: Greenhouse, Workday, SAP SuccessFactors integration

// Supabase Edge Function: create-checkout
import { stripe } from './stripe-client'

export async function POST(req) {
  const { priceId, userId } = await req.json()
  
  const session = await stripe.checkout.sessions.create({
    customer_email: user.email,
    line_items: [{ price: priceId, quantity: 1 }],
    mode: 'subscription',
    success_url: `${origin}/dashboard?success=true`,
    cancel_url: `${origin}/pricing`,
    metadata: { userId }
  })
  
  return new Response(JSON.stringify({ url: session.url }))
}

  • Target: Monthly churn < 5%

  • Fallback: Hybrid approach - deterministic rules + LLM only for edge cases
